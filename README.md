# hirop-orientdb

A [OrientDB](http://www.orientdb.org/) backend for [hirop](https://github.com/orobix/hirop).

## Usage

Leiningen:
```clojure
[hirop-orientdb "0.1.0-SNAPSHOT"]
```

Test:
```clojure
lein test
```

## Documentation

Please refer to the [hirop wiki](https://github.com/orobix/hirop/wiki).
